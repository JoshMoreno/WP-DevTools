# Requirements
PHP >= 7.1.3

WordPress >= 4.9

# Installation  
#### WP-CLI
```bash
wp plugin install --activate https://github.com/JoshMoreno/wpdev/releases/download/v1.2.1-alpha/wpdev.zip
```
#### Composer
```bash
composer require joshmoreno/wpdev
```

#### Manually
Nothing special. Just install and activate like any other plugin.
