Getting Started

Requirements

  • PHP 7+
  • WordPress 4.9+
  • Composer

Installation

WP-CLI
{% highlight shell %} # One Liner (copy all 4 lines) - Make sure you're directly in the plugins folder wp plugin install {{site.zipUrl}} && cd wpdev && composer install -o && wp plugin activate wpdev # --------------------------- # Download the plugin wp plugin install {{site.zipUrl}} # Move into wpdev folder cd plugins/wpdev # Install dependencies composer install -o # Activate wp plugin activate wpdev {% endhighlight %}
Manually
  1. Download the plugin and drop it into the plugins folder - latest release
  2. Run composer install -o via the command line inside the plugin folder.
  3. Activate
